## 📌 Project Overview

The damn-vulnerable-rust-app provides a safe space to practice security testing. It consists of a web application built in the Rust programming language. You use this tool to learn how hackers find weaknesses in software. Developers and students use these labs to fix common security gaps. The project includes real examples of flaws. You can study these flaws to build better programs.

## 💻 Requirements

Your computer must meet these basic needs to run the application:

*   A modern web browser like Chrome, Firefox, or Edge.

*   Administrator rights on your computer to install necessary components.

## 🛠️ Exploring the Labs

The application interface organizes security tasks into categories. Each category teaches you a specific way to test software.

### Web Server Security
This section focuses on how the server handles incoming requests. You learn why it is important to check data that users send to the page. You will see how the application reacts when it receives unexpected input.

### Data Handling
You explore how the software stores information. This lab teaches you to verify that data remains private and secure. You practice checking for gaps that allow unauthorized users to see your data.

### Automation and Testing
The application includes tools to help you test code faster. These tools show how developers scan their work for mistakes. You learn the value of finding errors before software reaches the public.

## 🔍 How to Learn

Spend time clicking through the menus. Read the descriptions provided for each lab. Each laboratory task includes instructions on what to look for. Start with the beginner labs. Do not worry if you fail to solve a puzzle immediately. Experiment with different inputs. The environment remains safe because it runs only on your computer.

## ⚙️ Troubleshooting

If you encounter issues, try these steps:

*   Check that no other program uses the same network port.
*   Restart your computer.
*   Ensure your firewall allows the application to communicate with your browser.
*   Download the latest version again if the installer crashes.
*   Check the black console window for error messages. If you see red text, the application may have failed to start. Close it and try again.

## 📖 Frequently Asked Questions

**Is this software dangerous for my computer?**
No. The application simulates vulnerabilities in a controlled environment. It does not contain real viruses or malware.

**Does the application send data over the internet?**
No. Every action happens on your local machine. Your progress stays on your computer.

**How do I delete the software?**
Open your Windows Settings, go to Apps, and find the application in the list. Select Uninstall to remove it from your system.

**Can I run this on a Mac or Linux?**
The current installer supports Windows. You may use Docker to run the application on other operating systems. See the project repository for more details on Docker workflows.
